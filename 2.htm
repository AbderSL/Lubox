How to install Lulubox plugins and mods

To install plugins to patch games with Lulubox, you must follow the following steps  :

1.    Step 1 : after opening the application, you must search between the available games that you want to patch and click on them.

2.    Step 2  : you have a menu on the attributes of the game that you can modify. It can be for example to increase the speed of the game, an infinite number of lives, increase the quantity of coins ... to choose the advantages with which you will count you must mark them as active and those which neither interest you, just turn them off.

3.    Step 3  : lastly, once you have cleared the aspects to patch, you must click on the Install button at the bottom of the screen. The plugin will begin to download and install, giving rise to modification of the game with its hacked functionality .

As you can see, even if Lulubox is used for the same thing as other applications like SB Game Hacker or Lucky Patcher, it works in a different and easier way for the user. Do not manually search for the values ​​except that it suffices to mark the attributes to be modified with patches created ad hoc for games.

From now on, the user should know that often applying these patches replaces the advantages he can obtain by using in-app purchases of the application by avoiding this process. These purchases are the developers' only income to monetize their product and, therefore, to earn money with their work.
